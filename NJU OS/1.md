#### 应用视角的操作系统

```shell
ls -l a.out		# 查看文件属性
file a.out		# 查看(猜测)文件类型
objdump -d a.out | less 
# 反汇编，打开二进制文件并找到所有汇编代码
objdump -d a.out | wc -l	# 查看行数，利用管道

gcc hello.c -static --verbose	
# 查看编译信息(运行日志)，寻找include pa

gcc -E hello.c	# 输出预编译结果
gcc -c hello.c	# 生成.o
ld hello.o		# 链接
ld hello.c -e main	# 指定程序入口
gcc hello.c -Wl,--verbose	#

# gdb
gdb a.out	# 调试
start 		# 从第一句开始
layout asm 	# UI界面
info registers 	# 寄存器状态
p $rsp		# 查看寄存器状态
x $rsp		# 查看寄存器内容
```



- 静态链接(150000)得到的文件比**动态链接(183行)**大很多，

- `include`将目标库代码直接复制过来

- ```
  .c (预处理)-> .l (cc)-> .s (as)-> .o (链接器)-> a.out
  ```
